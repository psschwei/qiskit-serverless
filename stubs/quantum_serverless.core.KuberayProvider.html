quantum_serverless.core.KuberayProvider¶

class KuberayProvider(name, host=None, namespace='default', img='icr.io/quantum-public/quantum-serverless-ray-node:latest-py39', token=None, compute_resource=None, available_compute_resources=None)[source]¶

Implements CRUD for Kuberay API server.

Kuberay provider for serverless computation.

Example

>>> provider = Provider(
>>>    name="<NAME>",
>>>    host="<HOST>",
>>>    namespace="<NAMESPACE>",
>>>    token="<TOKEN>",
>>>    compute_resource=ComputeResource(
>>>        name="<COMPUTE_RESOURCE_NAME>",
>>>        host="<COMPUTE_RESOURCE_HOST>"
>>>    ),
>>> )
Parameters:

  • name (str) – name of provider

  • host (Optional[str]) – host of provider a.k.a managers host

  • namespace (Optional[str]) – namespace to deploy provider in

  • image – container image to use for ray cluster

  • token (Optional[str]) – authentication token for manager

  • compute_resource (Optional[ComputeResource]) – selected compute_resource from provider

  • available_compute_resources (Optional[List[ComputeResource]]) – available clusters in provider
